# NBA Fantasy Simulator

A fantasy NBA sandbox web app built with **Next.js**, **React**, and **Tailwind CSS**.  
Users can create custom teams, compare lineups, and simulate weekly matchups using real NBA player stats.

## Features

- Browse active NBA players and their per-game stats.
- View last 5 games averages.
- Simulate matchups between custom teams.
- Lightweight JSON data preprocessed for frontend performance.

## Getting Started

1. **Install dependencies**
   `npm install`

2. **Run development server**
   `npm run dev`

## Notes

- JSON data is regenerated periodically and preprocessed on the frontend for performance.
- Built to be hosted on **Vercel**.
